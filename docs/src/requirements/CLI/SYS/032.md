# CLI-SYS-032 Graph Visualization Command

## Statement

The tool shall provide a 'req graph' command that generates visual representations of the requirements graph.

## Scope

This command creates visualizations of requirement relationships:

Output formats:
- ascii: Text-based tree visualization (default)
- dot: GraphViz DOT format
- svg: Scalable Vector Graphics (requires GraphViz)
- png: Portable Network Graphics (requires GraphViz)

Options:
- --direction: Graph direction (TB, BT, LR, RL)
- --depth: Limit visualization depth
- --output FILE: Write to file instead of stdout
- --open: Open visualization in default viewer

## Verification

- ASCII format displays tree structure in terminal
- DOT format generates valid GraphViz syntax
- SVG and PNG formats require GraphViz installation
- --direction flag controls graph layout orientation
- --depth flag limits recursion depth
- --output flag writes to specified file
- --open flag launches system default viewer
- Command can focus on subgraph from specific HRID(s)
- Exit codes: 0 (success), 1 (error, e.g., GraphViz not found)
