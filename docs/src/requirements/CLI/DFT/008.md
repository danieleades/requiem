# CLI-DFT-008 Sync command missing deprecated clean alias

## Observation
SYS-021/SYS-030 specify keeping the legacy `req clean` alias after renaming to `req sync`; the CLI only registers `req sync` with no `clean` alias.

## Evidence
- req/src/cli.rs lists `Sync` but no `Clean` variant; no alias wiring in command parsing.
- `rg "clean" req/src` finds no subcommand/alias implementation.

## Impact
Scripts and users relying on the prior `req clean` name break, violating the compatibility requirement.

## Expected
Provide a `req clean` alias (or deprecation shim) that maps to `req sync`, retaining behaviour and exit codes while warning about deprecation.
