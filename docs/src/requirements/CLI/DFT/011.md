# CLI-DFT-011 Graph visualization command missing

## Observation
The CLI has no `req graph` subcommand or equivalent implementation.

## Evidence
- req/src/cli.rs does not register a graph command; no graph module exists.
- SYS-032 requires the command with ascii/dot/svg/png outputs and related flags.

## Impact
Users cannot generate graph visualizations or export DOT/SVG/PNG as required by SYS-032.

## Expected
Provide the `req graph` command with the output formats and flags described in SYS-032.
