# MCP-SPC-002 Parameter Documentation Standards

## Statement
Every MCP tool parameter shall be documented so agents can pass correct values without guesswork.

## Standards
1) Brief description of what the parameter is used for.
2) Format or value constraints where applicable.
3) Example values (e.g., "USR" for kind, "USR-007" for HRID).
4) Whether the parameter is required or optional.
5) Special notes about case-sensitivity or normalization.
