# MCP-SPC-004 Recommended Tool Exploration Workflow

## Statement
Server instructions and tool descriptions shall guide agents toward a predictable exploration workflow.

## Workflow
1) `list_requirement_kinds` → understand what kinds exist.
2) `list_requirements` → find requirements by kind and query.
3) `get_requirement` → fetch full details including parents and children.
4) `get_children` → navigate the requirement graph.

This workflow should be mentioned in server instructions or tool descriptions.
