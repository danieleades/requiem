# SYS-026 Rename Requirement Command

## Statement

The tool shall provide a 'req rename' command that changes a requirement's HRID and updates all references.

## Scope

This command handles HRID renaming:
- Validates new HRID is available
- Updates first markdown heading with new HRID
- Renames file to match new HRID
- Optional --sync flag updates children's parent HRID references

This is distinct from move which changes location; rename changes the identifier.

## Verification

- Command updates HRID in markdown heading
- File renamed to match new HRID
- Validation prevents HRID conflicts
- --sync flag triggers children parent HRID updates
- --yes flag skips confirmation prompts
- Reports list of children that reference this requirement
