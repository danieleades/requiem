# SYS-031 Show Command

## Statement

The tool shall provide a 'req show' command that displays detailed information about a single requirement.

## Scope

This command provides a detailed view of one requirement, complementing the list command which shows multiple requirements in summary form.

Syntax: req show HRID [OPTIONS]

Displays:
- HRID, title, kind, namespace
- Creation and modification timestamps
- Parents (with suspect status indicators)
- Children  
- Tags
- File path
- Optionally: full markdown content

## Verification

- Command displays all metadata for specified requirement
- --edit flag opens requirement in EDITOR
- --output flag supports: pretty (default), json, markdown, raw
- --with-content flag includes full markdown body
- Pretty output is human-readable with clear formatting
- JSON output includes complete requirement structure
- Raw output shows original markdown file content
- Exit codes: 0 (success), 1 (requirement not found)
