# SYS-027 Unlink Command

## Statement

The tool shall provide a 'req unlink' command that removes parent-child links without manual frontmatter editing.

## Scope

This command removes a specific parent from a child's frontmatter. It is the inverse of 'req link'.

Syntax: req unlink CHILD PARENT

## Verification

- Command removes specified parent from child's frontmatter
- Validates both HRIDs exist before unlinking
- Validates link exists before attempting removal
- --yes flag skips confirmation prompt
- Exit codes: 0 (success), 1 (error, link does not exist), 130 (cancelled)
- Confirmation prompt shows child and parent HRIDs
